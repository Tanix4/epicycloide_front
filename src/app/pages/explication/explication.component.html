• Une épicycloïde est une courbe
  • plane transcendante, trajectoire d'un point fixé à un cercle qui
  • roule sans glisser sur un autre cercle dit directeur, les disques
  • roule sans glisser sur un autre cercle dit directeur, les disques
  • ouverts ayant ces deux cercles pour frontière étant disjoints.
  • Il s'agit donc d'un cas particulier de cycloïde à centre,
  • qui est une catégorie de courbe cycloïdale.

Une épicycloïde peut être définie par l'équation paramétrique suivante :

x = (R + r) * cos(t) - r * cos((R + r) * t / r)

y = (R + r) * sin(t) - r * sin((R + r) * t / r)

  • x et y sont les coordonnées du point sur l'épicycloïde.
  • R est le rayon du cercle fixe (épicyle).
  • r est le rayon du cercle roulant.
  • t est le paramètre qui varie de 0 à 2π.
